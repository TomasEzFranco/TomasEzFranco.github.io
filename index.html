Selected Projects

Thesis In Progress

LLM Guided Autonomous Pick And Place

Closed loop manipulation pipeline on the Quanser QArm using RGB depth sensing, YOLO instance segmentation, and an LLM planner. The system handles cube sorting and stacking with grasp checks, gain tuning, scene validation, and motion safety limits.

GitHub Portfolio Writeup

Autonomous Vehicles

MXCarkit Autonomous Driving Pipeline

Autonomous driving pipeline using Docker, PilotNet behavioral cloning, YOLOv11 traffic light detection, ROS2, Python, Foxglove validation, and live hardware testing on a small multi sensor vehicle platform.

Vision And Speech Pipeline

Healbot Robotic Assistant

Robotic assistant that interprets symptom focused speech requests, selects the correct item with an LLM, uses YOLO for object detection, and completes pick and place actions with the Quanser QArm.

ROMI Line Following Robot

Differential drive robot using IR sensing, wheel encoders, IMU feedback, heading control, and dead reckoning to complete a closed course with checkpoints and wall detection.
